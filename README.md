# 基于Python的校园二手交易与智能推荐系统
## 项目简介
本项目面向在校大学生打造校内闲置物品交易平台，实现账号注册登录、二手商品发布修改、商品关键词检索、下单交易、个性化好物智能推荐，解决教材、生活用品闲置浪费问题。

## 技术栈
编程语言：Python
前端页面：Streamlit
数据存储：本地文件存储

## 功能模块
1. 用户模块：注册账号、密码登录
2. 商品模块：发布二手商品、修改商品信息、下架已售商品
3. 检索推荐：关键词搜索商品、根据浏览记录智能推荐同类商品
4. 订单模块：选购商品、自动生成订单、订单信息查询

## 部署运行
1. 安装依赖：`pip install streamlit`
2. 项目启动：`streamlit run app.py`

## 项目不足与优化方向
现阶段无管理员后台、缺少用户聊天功能，后续迭代可补充以上功能。
